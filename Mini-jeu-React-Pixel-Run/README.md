# Mini-jeu-React-Pixel-Run
  Guide pour la répartition des tâches :-Dev 1 : logique du jeu (state, timer, collisions)-Dev 2 : affichage / animation (Framer Motion ou CSS 
transitions)-Dev 3 : score et écran de fin-Dev 4 : design général + thème
 Guide pour l’utilisation des Hooks  :
 useState → position du joueur, score, état du jeu (en cours / 
perdu).
 useEffect → boucle de jeu (rafraîchissement, collisions, 
timer).
 useContext → paramètres globaux (niveau, thème).
 useMemo → optimiser le calcul des collisions ou la
